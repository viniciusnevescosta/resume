## Quick Start

1. Clone repo:  
```bash
git clone https://github.com/yourusername/viniciusnevescosta.github.io.git
cd viniciusnevescosta.github.io
```

2. Start dev:  
```bash
pnpm dev
```

3. Deploy:  
```bash
pnpm github
```
